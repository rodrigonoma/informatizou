# Informatizou Creative System

## Princípios

### Identidade antes de estética

Toda escolha visual deve representar alguma característica real da empresa.

### Um conceito forte por projeto

Cada site deve possuir uma ideia central facilmente explicável em uma frase.

### Variação controlada

Os sites podem compartilhar padrões técnicos, mas não devem compartilhar a mesma composição visual.

### Conteúdo é estrutura

O layout deve nascer do conteúdo, não de um template pré-selecionado.

### Movimento com intenção

Animações devem comunicar hierarquia, continuidade, causalidade ou personalidade.

## Tipografia

Não utilizar automaticamente Inter, Roboto, Arial ou Poppins.

Escolher combinações coerentes com o posicionamento:

- serifada editorial + sans neutra;
- grotesca expressiva + monoespaçada;
- display condensada + sans humanista;
- serifada contemporânea + grotesca técnica.

Limitar-se preferencialmente a duas famílias tipográficas.

## Espaçamento

Usar uma escala consistente.

Exemplo:

- 4px;
- 8px;
- 12px;
- 16px;
- 24px;
- 32px;
- 48px;
- 64px;
- 96px;
- 128px.

## Cores

A paleta deve ser definida individualmente para cada cliente.

Utilizar:

- uma cor-base;
- uma cor de contraste;
- uma cor de ação;
- neutros;
- cores semânticas.

Não utilizar automaticamente a paleta da Informatizou nos sites dos clientes.

## Layout

Priorizar:

- hierarquia clara;
- tensão visual controlada;
- variação de escala;
- uso intencional de espaço negativo;
- grids flexíveis;
- quebras de grid justificadas;
- alternância de densidade;
- composição responsiva.
